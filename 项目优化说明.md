# Chat Server 项目说明

## 项目概述

这是一个即时通讯项目底座，后端使用 FastAPI + SQLite，前端使用 React + Vite。项目启动后不会自动创建内置账号，用户需要自行注册账号后使用。

## 已实现能力

- 注册、登录、退出
- Session token 认证
- 好友申请、同意、拒绝
- 按账号搜索并申请好友
- 好友通过后才能建立私聊
- 私聊、群聊、群成员添加
- 会话置顶、静音、归档、删除
- 消息发送、编辑、撤回
- 幂等 `clientId`
- 已读/送达回执、未读同步
- 消息收藏
- 通知中心
- 审计日志
- 会话和消息搜索
- AI 摘要、关键词和待办提取
- 健康检查和运行指标
- SSE 实时更新和在线状态

## 目录结构

```text
app/                  FastAPI 后端
frontend/             React/Vite 前端
scripts/smoke_test.py 后端核心链路烟测
requirements.txt      Python 依赖
README.md             GitHub 项目说明
```

本地数据库、日志、构建产物、依赖目录和截图输出均已写入 `.gitignore`。

## 运行方式

后端：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

前端开发：

```bash
cd frontend
npm install
npm run dev -- --host 0.0.0.0 --port 5173
```

生产构建：

```bash
cd frontend
npm install
npm run build
cd ..
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## 验证命令

```bash
python3 -m compileall app scripts
python3 scripts/smoke_test.py
cd frontend && npm run build
```

`scripts/smoke_test.py` 使用临时 SQLite 数据库，验证空库初始化、注册、好友申请、同意、私聊、发消息、收藏、搜索和摘要，不会污染本地运行数据库。
