# Chat Server 项目说明

## 项目概述

这是一个即时通讯项目底座，后端使用 C++17 实现，前端使用 React + Vite。项目启动后不会自动创建内置账号，用户需要自行注册账号后使用。

## 已实现能力

- 注册、登录、退出
- Session token 认证
- 好友申请、同意、拒绝
- 按账号搜索并申请好友
- 好友通过后才能建立私聊
- 私聊、群聊、群成员添加
- 会话置顶、静音、归档、删除
- 消息发送、编辑、撤回
- 幂等 `clientId`
- 已读/送达回执、未读同步
- 消息收藏
- 通知中心
- 审计日志
- 会话和消息搜索
- AI 摘要、关键词和待办提取
- 健康检查和运行指标
- SSE 兼容端点和在线状态

## 目录结构

```text
cpp_server/              C++17 后端
frontend/                React/Vite 前端
scripts/smoke_test_cpp.mjs 后端核心链路烟测
Makefile                 g++ 构建入口
CMakeLists.txt           CMake 构建入口
README.md                GitHub 项目说明
```

本地数据、日志、构建产物、依赖目录和截图输出均已写入 `.gitignore`。

## 运行方式

后端：

```bash
make
make run
```

前端开发：

```bash
cd frontend
npm install
npm run dev -- --host 0.0.0.0 --port 5173
```

生产构建：

```bash
cd frontend
npm install
npm run build
cd ..
make run
```

## 验证命令

```bash
make
BASE_URL=http://127.0.0.1:8000 node scripts/smoke_test_cpp.mjs
cd frontend && npm run build
```

`scripts/smoke_test_cpp.mjs` 验证注册、好友申请、同意、私聊、发消息、收藏、搜索和摘要。需要避免污染正式数据时，可先用临时 `DATA_FILE` 启动 C++ 服务。
